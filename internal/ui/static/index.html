Wildberries Storage MVP

Панель проверки бэкенда

Здесь можно руками проверить прогноз, решение по вызову транспорта, batch-генерацию, метрики и выбор модели.

/predict

Отправляет точки во внешний ML-сервис через Go API.

/decision

Принимает либо точки, либо `predictions` из `/predict` плюс исходные `points`.

/batch

Генерирует `submission.csv` и может вернуть прогнозы в ответе.

/metrics

Считает `WAPE + |Relative Bias|` прямо из JSON-наблюдений.

/model/select

Проксирует выбор модели в ML-сервис.
